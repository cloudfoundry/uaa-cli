# list-group-mappings

List all mappings between UAA groups (scopes) and external groups from identity providers.

## Usage

```
uaa list-group-mappings
```

## Global Flags

| Flag | Short | Description |
|------|-------|-------------|
| `--verbose` | `-v` | Print additional info on HTTP requests |

## Examples

```bash
uaa list-group-mappings
```
