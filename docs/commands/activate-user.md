# activate-user

Activate a previously deactivated user account by username.

## Usage

```
uaa activate-user USERNAME
```

## Global Flags

| Flag | Short | Description |
|------|-------|-------------|
| `--verbose` | `-v` | Print additional info on HTTP requests |

## Examples

```bash
uaa activate-user bob
```
